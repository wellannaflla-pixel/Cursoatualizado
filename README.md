# 🛡️ Navegação Segura | Hub de Proteção Digital

> **"Conhecimento é a melhor proteção."**

O **Navegação Segura** é uma plataforma educacional desenvolvida com o objetivo de promover a conscientização digital. O projeto centraliza informações críticas sobre ameaças virtuais, orientações de controle parental e canais de denúncia, tudo em uma interface moderna, responsiva e de alta performance.

---

## 🚀 O que há de novo na v2.0?

Nesta atualização, foquei em **experiência do usuário (UX)** e **design de interface (UI)**:

- **Animações Dinâmicas:** Integração com a biblioteca *AOS (Animate On Scroll)* para uma navegação fluida e envolvente.
- **Micro-interações:** Cards interativos com efeitos de sombra e elevação que melhoram a percepção de qualidade do software.
- **Arquitetura de Informação:** Nova seção de checklist (Dicas Rápidas) para facilitar a leitura e o aprendizado sobre cibersegurança.
- **Otimização SEO:** Configuração de Meta Tags e Favicons para garantir uma apresentação profissional em redes sociais.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as tecnologias padrão da indústria para garantir rapidez e manutenibilidade:

* **Frontend:** HTML5, CSS3, JavaScript.
* **UI Framework:** Bootstrap 5.3 (Grid System e Componentes).
* **Animações:** AOS Library.
* **Ícones:** Bootstrap Icons.
* **Tipografia:** Google Fonts (Inter).
